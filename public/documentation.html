Movies API

Hi! welcome to my Marvel Movies API created by Colette Parker.
This is a REST API connected to a database that stores data about different marvel movies.

Movies API actions

Description URL HTTP Method Request body data format Response body data format
Return a list of movies to the user /movies Get none A JSON object containing all movies
Return data about a single movie by title to the user /movies/:Title Get none A JSON object containing data, including title, img, genre, description and director of a single movie by title 
                                                    
                                                        {
                                                        id: '',
                                                        title: '',
                                                        description: '',
                                                        genre: '',
                                                        director: '',
                                                        img: ''
                                                        }
Return data about a genre by name /movies/genre/:Name Get none A JSON object containing data about the genre: 
                                                    
                                                        {
                                                        genre: '',
                                                        description: ''
                                                        }
Return data about a director /movies/directors/:Name Get none A JSON object containing data about the director: 
                                                    
                                                        {
                                                        name: '',
                                                        biography: '',
                                                        date of birth: '',
                                                        date of death: ''
                                                        }
Allow new user to register /users Post A JSON object containing data about the user to add, structured so: 
                                                    
                                                        {
                                                        username: '',
                                                        password: '',
                                                        email: '',
                                                        birthday: '',
                                                        favoriteMovies: []
                                                        }
A JSON object containing data about the users added, inluding an ID: 
                                                    
                                                        {
                                                        id: '',
                                                        username: '',
                                                        password:'',
                                                        email: '',
                                                        birthday: '',
                                                        favoriteMovies: []
                                                        }
Allow users to update their info /users/:Username Put none 
                                                    
                                                        {
                                                        id: '',
                                                        username: '',
                                                        password: '',
                                                        email: '',
                                                        birthday: '',
                                                        favoriteMovies: []
                                                        }
Allow users to add a movie to list of favorites /users/:username/movies/:MovieID Post none A message confirming that a movie has been added to favourites
Allow users to remove a movie from list of favorites /users/:Username/movies/:MovieID Delete none A message confirming that a movie has been removed from favourites
Allow existing users to deregister /users/:Username Delete none A message confirming user was successfully deregistered
